# FsDateFormat

> 日期格式化展示组件

## Props

| Prop name   | Description                                         | Type   | Values | Default               |
| ----------- | --------------------------------------------------- | ------ | ------ | --------------------- |
| modelValue  | 日期时间值，支持 long,string,date 等，由 dayjs 转化 | -      | -      |                       |
| valueFormat | 输入格式化，不传则由 dayjs 自动转化                 | string | -      | undefined             |
| format      | 输出格式化                                          | string | -      | "YYYY-MM-DD HH:mm:ss" |
